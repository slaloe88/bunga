Dalam contoh di atas, kita menggunakan elemen
untuk membuat bunga. Kita menggunakan CSS untuk memberikan tampilan bunga dengan memanfaatkan properti seperti background-color, border-radius, dan transform. Setiap kelopak bunga diwakili oleh elemen
dengan kelas petal, dan kita menggunakan properti transform: rotate() untuk memutar kelopak bunga sesuai dengan sudut yang diinginkan. Di sisi JavaScript, kita menggunakan getElementsByClassName() untuk mendapatkan semua elemen dengan kelas petal. Kemudian, kita menggunakan perulangan untuk mengubah properti transform dari setiap kelopak bunga dengan menambahkan sudut rotasi angle. Kita menggunakan requestAnimationFrame() untuk melakukan animasi rotasi bunga dengan kecepatan rotationSpeed. Untuk melihat bunga dalam aksi, Anda dapat menyimpan kode di atas dalam file HTML, lalu membukanya melalui browser. Anda akan melihat bunga dengan kelopak yang berputar secara animasi. Anda dapat mengkustomisasi kode ini dengan menambahkan lebih banyak kelopak, mengubah warna, ukuran, atau mengatur kecepatan rotasi sesuai preferensi Anda.
